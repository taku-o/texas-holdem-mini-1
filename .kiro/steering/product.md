# Product Overview

テキサスホールデムのブラウザ向けWebアプリケーション。人間1人＋CPU4人＋ディーラーCPU1人の計6人でプレイするシングルプレイヤー型ポーカーゲーム。

## Core Capabilities

- テキサスホールデムの標準ルールに沿ったゲーム進行（ブラインド→プリフロップ→フロップ→ターン→リバー→ショーダウン）
- 1セッション完結型：リロードでリセット、永続化なし
- Apple公式アプリ風のクリーンなUI（階層・余白・タイポグラフィ重視）

## Target Use Cases

- ブラウザで手軽にテキサスホールデムを遊びたいユーザー
- CPU相手の1セッション単位のプレイ

## Value Proposition

- バックエンド・DB不要のクライアント完結型
- 必要最小限の機能に絞った実装

## Scope Boundaries

- マルチプレイヤー・ログイン・永続化・ランキングは対象外
- 高度なCPU戦略（学習・AI）は対象外
- サイドポットは扱わない（単一ポット）
- リアルマネー・課金は対象外

# Product Overview

テキサスホールデムのミニ版を扱うプロジェクト。ゲームルールを簡略化した小規模なポーカーアプリを想定する。

## Core Capabilities

- テキサスホールデムの基本ルールに沿ったゲーム進行
- 少人数・短時間で遊べるミニ版としての体験
- （仕様策定時に `.kiro/specs/` で具体化する）

## Target Use Cases

- 手軽にポーカールールを試したいユーザー
- 少人数・短時間の1ゲーム単位のプレイ

## Value Proposition

- ルールがシンプルで理解しやすいミニ版
- 必要最小限の機能に絞った実装
